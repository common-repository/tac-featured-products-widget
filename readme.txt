Creates a featured products widget for use with WP e-Commerce plug in

== Description ==
=It displays up to 9 featured products from a featured products category in a widget.   This works with WP e-Commerce Plugin already installed.  You must create a category called Featured.  The number of featured products shown in the widget depends on how many featured delete that products you have in the Featured category.=

== Installation ==

1.  Automatically install TAC Featured Products Widget using add new search from your admin.
2.  Make sure WP e-Commerce plug in is activated and working correctly.
3.  Activate the TAC Featured Products Widget through the Plugins menu in WordPress dashboard
4.  Appearance and then Widgets on your word press dashboard
